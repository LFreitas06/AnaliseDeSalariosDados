📊 Dashboard de Análise de Salários na Área de Dados
Este projeto é uma aplicação interativa desenvolvida para explorar e analisar tendências salariais no mercado de dados global. Através de uma interface visual, o utilizador pode filtrar informações por senioridade, tipo de contrato, ano e tamanho da empresa.

🔗 Link de Acesso
Pode aceder ao dashboard em tempo real aqui: https://projetodadospython.streamlit.app/

🚀 Funcionalidades
Métricas: Visualização do salário médio, salário máximo, total de registos e o cargo mais frequente.

Gráficos Interativos: Top 10 Cargos, Distribuição Salarial, Proporção de Trabalho Remoto e um Mapa Global para Cientistas de Dados.

Filtros Dinâmicos: Barra lateral para ajustar a análise por Ano, Senioridade, Tipo de Contrato e Tamanho da Empresa.

Exploração de Dados: Tabela detalhada para consulta individual dos registos filtrados.

🛠️ Tecnologias Utilizadas

Streamlit: Interface web.

Pandas: Manipulação e limpeza de dados.

Plotly: Gráficos dinâmicos.

📁 Estrutura do Projeto

app.py: Código principal da aplicação.

requirements.txt: Dependências do projeto.

dados-imersao-final.csv: Base de dados utilizada.

🔧 Como Executar Localmente

Clone o repositório.

Instale as dependências: pip install -r requirements.txt.

Execute: streamlit run app.py.
